### Integratsioonid veebis:

- **E-posti turundus**: Mailchimp / ConvertKit
- **Kalender**: Google Calendar API
- **Piletid**: Piletilevi, Ticketmaster
- **Muusika**: Spotify, Apple Music, YouTube
- **Sotsiaalmeedia**: Instagram, Facebook, TikTok API
- **Analüütika**: Google Analytics 4, Hotjar
- **CRM**: broneeringute haldamiseks
- **Maksed**: kui on plaanis epood veebimüük 

### Engagement метрики:
- Количество просмотров видео
- Подписки на рассылку (цель: 50+ в месяц)
- Социальные расшаривания
- Загрузки пресс-кита


### Детальная структура страниц:

#### **Главная страница**
- Hero-секция с ярким видео выступления (10-15 сек)
- Слоган: "Функ, который заставляет танцевать"
- CTA: "Послушать музыку" / "Забронировать группу"
- Ближайшие концерты (3-4 события)
- Отзывы клиентов (карусель)
- Видео-тизеры лучших моментов
- Логотипы компаний-клиентов
- Интеграция с Instagram feed

#### **О группе**
- История создания (таймлайн 2017-2024)
- Философия и стиль
- Состав группы (фото + биография каждого музыканта)
- Достижения и награды
- Коллаборации
- "В прессе о нас" (цитаты из обзоров)

#### **Музыка**
- Embedded плееры Spotify/Apple Music
- Альбомы с обложками:
  - "We Need To Talk" (2019) - треклист + описание
  - "Blind Date" (2021) - треклист + описание
- Видеоклипы
- Ссылки на стриминговые платформы
- Возможность покупки физических копий

#### **Концерты/События**
- Календарь предстоящих выступлений
- Фильтр по типу события (фестиваль/клуб/частное мероприятие)
- Интеграция с Google Calendar
- "Прошедшие концерты" с фото/видео
- Форма "Пригласить на мероприятие"

#### **Бронирование**
- Типы мероприятий (корпоратив, свадьба, фестиваль, etc.)
- Форма запроса с полями:
  - Тип события
  - Дата и место
  - Ожидаемое количество гостей
  - Бюджет (диапазон)
  - Особые пожелания
  - Контактные данные
- FAQ по бронированию
- Технический райдер (скачать PDF)
- Процесс работы (5 шагов)

#### **Галерея**
- Профессиональные фото выступлений (сетка)
- Видео (полные концерты и клипы)
- За кулисами
- Фильтры по году/месту

#### **Медиа/Пресса** (новая страница)
- Пресс-релизы
- Пресс-кит для скачивания:
  - Hi-res фото
  - Биографии
  - Логотип группы
  - Технический райдер
- Контакт для СМИ
- Обзоры и статьи о группе

#### **Контакты**
- Контактная форма
- Email, телефон
- Социальные сети (иконки с ссылками)
- Менеджмент/букинг контакты
- Форма подписки на рассылку
