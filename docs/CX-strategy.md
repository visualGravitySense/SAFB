# CX План: Обновление сайта Siim Aimla Funk Band

## 1. Анализ текущего состояния

### Сильные стороны:
- Профессиональные фотографии высокого качества
- Четкая структура контента (О группе, Альбомы, Контакты)
- Эмоциональный и энергичный текст, передающий дух группы
- Упоминание коллабораций с известными вокалистами

### Проблемы и возможности для улучшения:
- Отсутствие календаря концертов/выступлений
- Нет видеоконтента или аудиосэмплов
- Ограниченная информация об альбомах
- Нет прямой возможности бронирования группы
- Отсутствие социальных доказательств (отзывы, пресса)
- Слабая интеграция с соцсетями

## 2. Целевая аудитория

### Основные сегменты:

**Сегмент 1: Организаторы мероприятий**
- Корпоративные event-менеджеры
- Организаторы свадеб и частных праздников
- Фестивальные директоры
- **Потребности**: быстрая оценка группы, видео выступлений, технический райдер, процесс бронирования

**Сегмент 2: Поклонники музыки**
- Любители джаза и фанка
- Посетители концертов
- Коллекционеры музыки
- **Потребности**: расписание концертов, прослушивание музыки, покупка билетов/альбомов

**Сегмент 3: Музыкальная индустрия**
- Журналисты
- Промоутеры
- Агенты
- **Потребности**: пресс-кит, фото высокого разрешения, биография, контакты для СМИ

## 3. Customer Journey Map

### Путь 1: Организатор мероприятия

**Этап 1: Осведомленность**
- Поиск в Google "funk band Estonia", "live band for event"
- Рекомендация от коллеги
- **Touchpoint**: SEO, реклама, сарафанное радио

**Этап 2: Рассмотрение**
- Посещение сайта
- Просмотр видео выступлений
- Оценка стиля музыки
- **Touchpoint**: Главная страница, видеогалерея, страница "Repertoire"
- **Pain point**: Нет видео, сложно оценить уровень

**Этап 3: Решение**
- Проверка доступности
- Запрос коммерческого предложения
- **Touchpoint**: Форма бронирования, контактная страница
- **Pain point**: Нет информации о ценах, непонятно как забронировать

**Этап 4: Бронирование**
- Согласование деталей
- Подписание договора
- **Touchpoint**: Email, телефон
- **Opportunity**: Автоматизация через онлайн-форму

**Этап 5: После мероприятия**
- Обратная связь
- Повторное бронирование
- **Touchpoint**: Follow-up email, отзывы
- **Opportunity**: Программа лояльности для повторных клиентов

### Путь 2: Поклонник музыки

**Этап 1: Открытие**
- Реклама концерта
- Видео в социальных сетях
- Рекомендация друга
- **Touchpoint**: Instagram, Facebook, Spotify

**Этап 2: Интерес**
- Прослушивание музыки
- Чтение о группе
- **Touchpoint**: Сайт, стриминговые платформы
- **Pain point**: Нет возможности послушать на сайте

**Этап 3: Вовлечение**
- Подписка на новости
- Покупка билета на концерт
- **Touchpoint**: Newsletter signup, билетная система
- **Opportunity**: Email-маркетинг с анонсами

**Этап 4: Посещение концерта**
- Живое выступление
- Покупка мерча
- **Touchpoint**: Концертная площадка

**Этап 5: Лояльность**
- Повторные визиты
- Покупка альбомов
- Распространение информации
- **Touchpoint**: Социальные сети, сайт группы
- **Opportunity**: Fan club, эксклюзивный контент

## 4. Ключевые метрики успеха (KPIs)

### Бизнес-метрики:
- Количество запросов на бронирование (цель: +40% за 6 месяцев)
- Конверсия посетителей в запросы (цель: 5-8%)
- Средняя стоимость контракта
- Количество повторных бронирований

### Пользовательские метрики:
- Время на сайте (цель: 3+ минуты)
- Показатель отказов (цель: <40%)
- Страниц за сессию (цель: 4+)
- Процент мобильного трафика

### Engagement метрики:
- Количество просмотров видео
- Подписки на рассылку (цель: 50+ в месяц)
- Социальные расшаривания
- Загрузки пресс-кита

## 5. Структура обновленного сайта

### Навигация:
```
ГЛАВНАЯ | О ГРУППЕ | МУЗЫКА | КОНЦЕРТЫ | БРОНИРОВАНИЕ | ГАЛЕРЕЯ | КОНТАКТЫ
```

### Детальная структура страниц:

#### **Главная страница**
- Hero-секция с ярким видео выступления (10-15 сек)
- Слоган: "Функ, который заставляет танцевать"
- CTA: "Послушать музыку" / "Забронировать группу"
- Ближайшие концерты (3-4 события)
- Отзывы клиентов (карусель)
- Видео-тизеры лучших моментов
- Логотипы компаний-клиентов
- Интеграция с Instagram feed

#### **О группе**
- История создания (таймлайн 2017-2024)
- Философия и стиль
- Состав группы (фото + биография каждого музыканта)
- Достижения и награды
- Коллаборации
- "В прессе о нас" (цитаты из обзоров)

#### **Музыка**
- Embedded плееры Spotify/Apple Music
- Альбомы с обложками:
  - "We Need To Talk" (2019) - треклист + описание
  - "Blind Date" (2021) - треклист + описание
- Видеоклипы
- Ссылки на стриминговые платформы
- Возможность покупки физических копий

#### **Концерты/События**
- Календарь предстоящих выступлений
- Фильтр по типу события (фестиваль/клуб/частное мероприятие)
- Интеграция с Google Calendar
- "Прошедшие концерты" с фото/видео
- Форма "Пригласить на мероприятие"

#### **Бронирование**
- Типы мероприятий (корпоратив, свадьба, фестиваль, etc.)
- Форма запроса с полями:
  - Тип события
  - Дата и место
  - Ожидаемое количество гостей
  - Бюджет (диапазон)
  - Особые пожелания
  - Контактные данные
- FAQ по бронированию
- Технический райдер (скачать PDF)
- Процесс работы (5 шагов)

#### **Галерея**
- Профессиональные фото выступлений (сетка)
- Видео (полные концерты и клипы)
- За кулисами
- Фильтры по году/месту

#### **Медиа/Пресса** (новая страница)
- Пресс-релизы
- Пресс-кит для скачивания:
  - Hi-res фото
  - Биографии
  - Логотип группы
  - Технический райдер
- Контакт для СМИ
- Обзоры и статьи о группе

#### **Контакты**
- Контактная форма
- Email, телефон
- Социальные сети (иконки с ссылками)
- Менеджмент/букинг контакты
- Форма подписки на рассылку

## 7. Контент-стратегия

### Типы контента:

**Видео** (приоритет #1):
- Hero video (15 сек) - лучшие моменты
- Полные концертные записи (3-5 песен)
- Короткие клипы для соцсетей (30-60 сек)
- Backstage видео
- Видео-отзывы клиентов

**Аудио**:
- Embedded Spotify плейлист "Best of Siim Aimla Funk Band"
- Сэмплы с каждого альбома
- Live recordings

**Тексты**:
- Расширенные биографии музыкантов
- Истории за песнями
- Блог/новости о группе
- Case studies успешных мероприятий

**Визуалы**:
- Профессиональная фотосессия (100+ фото)
- Инфографика: "Как подготовиться к выступлению группы"
- Графики: эволюция группы, карта выступлений

**Социальные доказательства**:
- Отзывы клиентов (текст + видео)
- Цитаты из музыкальной прессы
- Логотипы компаний-клиентов
- Цифры: 200+ мероприятий, 50,000+ зрителей, etc.

## 8. Технические требования

- CMS: Squarespace (текущая) или WordPress для большей гибкости

- Регулярные бэкапы

- **Платежи**: если планируется онлайн-продажа

### SEO:
- Оптимизация под ключевые слова:
  - "funk band Estonia"
  - "live band for corporate event"
  - "jazz band Tallinn"
  - "wedding band Estonia"
- Локальное SEO (Google My Business)
- Schema markup для событий
- Sitemap XML
- Оптимизация скорости загрузки

## 11. Риски и митигация

### Риск 1: Низкая конверсия запросов на бронирование
**Митигация**: 
- Упростить форму бронирования (минимум полей)
- Добавить live chat
- A/B тестирование CTA кнопок
- Добавить больше социальных доказательств

### Риск 2: Медленная загрузка из-за видео
**Митигация**:
- Использовать CDN
- Оптимизировать размер видео файлов
- Lazy loading для медиа
- Адаптивные изображения

### Риск 3: Недостаточно контента для поддержания актуальности
**Митигация**:
- Контент-календарь (еженедельные обновления)
- Интеграция соцсетей для автообновления
- Блог с регулярными постами
- Пользовательский контент (фото фанатов с концертов)

### Риск 4: Низкий органический трафик
**Митигация**:
- Комплексная SEO стратегия
- Контент-маркетинг
- Социальные сети
- Партнерство с event-платформами
- Google Ads для начального буста

## 14. Ключевые рекомендации

### ТОП-5 приоритетов:

1. **ВИДЕО, ВИДЕО, ВИДЕО** 
   - Это музыкальная группа - люди должны видеть и слышать выступления. Инвестируйте в качественную съемку.

2. **Упростите путь к букингу**
   - Каждая страница должна иметь явный CTA "Забронировать группу". Форма должна быть простой.

3. **Социальные доказательства**
   - Отзывы клиентов, логотипы компаний, цитаты из прессы - это критично для доверия.

4. **Мобильный first**
   - Большинство людей будут искать группу на мобильном. Дизайн должен быть идеальным на смартфонах.

5. **Регулярное обновление контента**
   - Сайт должен быть живым: новые фото, анонсы концертов, новости группы.

### Что НЕ делать:
- ❌ Не перегружайте сайт информацией
- ❌ Не делайте сложную навигацию
- ❌ Не прячьте контактную информацию
- ❌ Не используйте стоковые фото
- ❌ Не запускайте без мобильной оптимизации
